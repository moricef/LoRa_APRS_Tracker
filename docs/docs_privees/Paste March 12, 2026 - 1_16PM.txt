                                                                                                                                                                                                                                                                                           
 ### 1. Analyse (Corrigée)                                                                                                                                                                                                                                                                  
                                                                                                                                                                                                                                                                                            
 L'objectif est de déplacer uniquement la "physique" du toucher dans le contrôleur, tout en laissant la "mécanique" du rendu (déplacement de la carte, changement de tuile) dans ui_map_manager.cpp.                                                                                        
                                                                                                                                                                                                                                                                                            
 Flux de Données :                                                                                                                                                                                                                                                                          
 - ui_map_manager -> touchCtrl : Envoie les événements bruts (x, y, temps).                                                                                                                                                                                                                 
 - touchCtrl -> ui_map_manager : Expose son état interne via des getters (getVelocityX, isDragging, etc.).                                                                                                                                                                                  
                                                                                                                                                                                                                                                                                            
 ui_map_manager.cpp reste le seul maître de la logique de rendu (appel à scrollMap) et de la gestion des événements spécifiques (clics sur les stations, double-tap).

 Fichier : src/map_touch_controller.cpp                                                                                                                                                                                                                                                     
                                                                                                                                                                                                                                                                                            
 ```cpp                                                                                                                                                                                                                                                                                     
   #include "map_touch_controller.h"                                                                                                                                                                                                                                                        
   #include <cmath> // for fabsf                                                                                                                                                                                                                                                            
                                                                                                                                                                                                                                                                                            
   MapTouchController::MapTouchController()                                                                                                                                                                                                                                                 
       : _isScrollingMap(false),                                                                                                                                                                                                                                                            
         _dragStarted(false),                                                                                                                                                                                                                                                               
         _wasDragging(false),                                                                                                                                                                                                                                                               
         _breakFollow(false),                                                                                                                                                                                                                                                               
         _velocityX(0.0f),                                                                                                                                                                                                                                                                  
         _velocityY(0.0f),                                                                                                                                                                                                                                                                  
         _last_x(0),                                                                                                                                                                                                                                                                        
         _last_y(0),                                                                                                                                                                                                                                                                        
         _last_time(0)                                                                                                                                                                                                                                                                      
   {                                                                                                                                                                                                                                                                                        
   }                                                                                                                                                                                                                                                                                        
                                                                                                                                                                                                                                                                                            
   void MapTouchController::onTouchPressed(int x, int y, uint32_t current_time_ms) {                                                                                                                                                                                                        
       _last_x = x;                                                                                                                                                                                                                                                                         
       _last_y = y;                                                                                                                                                                                                                                                                         
       _last_time = current_time_ms;                                                                                                                                                                                                                                                        
       _dragStarted = false;                                                                                                                                                                                                                                                                
       _wasDragging = false;                                                                                                                                                                                                                                                                
       _isScrollingMap = true;                                                                                                                                                                                                                                                              
       _velocityX = 0.0f;                                                                                                                                                                                                                                                                   
       _velocityY = 0.0f;                                                                                                                                                                                                                                                                   
       _breakFollow = false;                                                                                                                                                                                                                                                                
   }                                                                                                                                                                                                                                                                                        
                                                                                                                                                                                                                                                                                            
   void MapTouchController::onTouchPressing(int x, int y, uint32_t current_time_ms) {                                                                                                                                                                                                       
       if (!_isScrollingMap) return;                                                                                                                                                                                                                                                        
                                                                                                                                                                                                                                                                                            
       int dx = x - _last_x;                                                                                                                                                                                                                                                                
       int dy = y - _last_y;                                                                                                                                                                                                                                                                
       uint32_t dt = current_time_ms - _last_time;                                                                                                                                                                                                                                          
                                                                                                                                                                                                                                                                                            
       if (!_dragStarted) {                                                                                                                                                                                                                                                                 
           if (abs(dx) > START_THRESHOLD || abs(dy) > START_THRESHOLD) {                                                                                                                                                                                                                    
               _dragStarted = true;                                                                                                                                                                                                                                                         
               _breakFollow = true; // Un drag manuel a été initié                                                                                                                                                                                                                          
           }                                                                                                                                                                                                                                                                                
       }                                                                                                                                                                                                                                                                                    
                                                                                                                                                                                                                                                                                            
       if (_dragStarted && dt > 0) {                                                                                                                                                                                                                                                        
           float weight = 0.7f;                                                                                                                                                                                                                                                             
           _velocityX = _velocityX * (1.0f - weight) + (-(float)dx / (float)dt) * weight;                                                                                                                                                                                                   
           _velocityY = _velocityY * (1.0f - weight) + (-(float)dy / (float)dt) * weight;                                                                                                                                                                                                   
       }                                                                                                                                                                                                                                                                                    
                                                                                                                                                                                                                                                                                            
       _last_x = x;                                                                                                                                                                                                                                                                         
       _last_y = y;                                                                                                                                                                                                                                                                         
       _last_time = current_time_ms;                                                                                                                                                                                                                                                        
   }                                                                                                                                                                                                                                                                                        
                                                                                                                                                                                                                                                                                            
   void MapTouchController::onTouchReleased() {                                                                                                                                                                                                                                             
       _wasDragging = _dragStarted;                                                                                                                                                                                                                                                         
       _isScrollingMap = false;                                                                                                                                                                                                                                                             
       _dragStarted = false;                                                                                                                                                                                                                                                                
                                                                                                                                                                                                                                                                                            
       if (fabsf(_velocityX) < 0.1f) _velocityX = 0.0f;                                                                                                                                                                                                                                     
       if (fabsf(_velocityY) < 0.1f) _velocityY = 0.0f;                                                                                                                                                                                                                                     
   }                                                                                                                                                                                                                                                                                        
                                                                                                                                                                                                                                                                                            
   void MapTouchController::updateInertia(uint32_t current_time_ms, bool isRedrawing) {                                                                                                                                                                                                     
       if (!_isScrollingMap && (_velocityX != 0.0f || _velocityY != 0.0f)) {                                                                                                                                                                                                                
           _last_time = current_time_ms;                                                                                                                                                                                                                                                    
           float friction = isRedrawing ? PAN_FRICTION_BUSY : PAN_FRICTION;                                                                                                                                                                                                                 
           _velocityX *= friction;                                                                                                                                                                                                                                                          
           _velocityY *= friction;                                                                                                                                                                                                                                                          
                                                                                                                                                                                                                                                                                            
           if (fabsf(_velocityX) < 0.01f) _velocityX = 0.0f;                                                                                                                                                                                                                                
           if (fabsf(_velocityY) < 0.01f) _velocityY = 0.0f;                                                                                                                                                                                                                                
       }                                                                                                                                                                                                                                                                                    
   }                                                                                                                                                                                                                                                                                        
                                                                                                                                                                                                                                                                                            
   // Getters                                                                                                                                                                                                                                                                               
   float MapTouchController::getVelocityX() const { return _velocityX; }                                                                                                                                                                                                                    
   float MapTouchController::getVelocityY() const { return _velocityY; }                                                                                                                                                                                                                    
   bool MapTouchController::isDragging() const { return _dragStarted; }                                                                                                                                                                                                                     
   bool MapTouchController::wasDragging() const { return _wasDragging; }                                                                                                                                                                                                                    
   bool MapTouchController::shouldBreakFollow() const { return _breakFollow; }                                                                                                                                                                                                              
   int MapTouchController::getLastX() const { return _last_x; }                                                                                                                                                                                                                             
   int MapTouchController::getLastY() const { return _last_y; }                                                                                                                                                                                                                             
   uint32_t MapTouchController::getLastTime() const { return _last_time; }                                                                                                                                                                                                                  
   bool MapTouchController::isScrolling() const { return _isScrollingMap; }                                                                                                                                                                                                                 
 ```
